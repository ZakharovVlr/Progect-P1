# 🎨 Impasto — веб-фоторедактор

Адаптивный и современный одностраничный сайт (Landing Page), созданный с использованием современных практик веб-разработки. 


## 🚀 Демонстрация
Посмотреть работающий сайт вживую можно по ссылке:
👉 [https://zakharovvlr.github.io/Progect-P1/]

## 🛠️ Стек технологий
* **HTML5** — семантическая структура и разметка.
* **CSS3** — стилизация, Flexbox/Grid для сетки, кастомные переменные и CSS-анимации.
* **JavaScript** — интерактивность и логика работы элементов.

## 📦 Структура проекта
```text
├── fonts/          # Локальные шрифты
├── images/         # Растровые изображения (PNG, JPG)
├── svg/            # Векторная графика
├── styles/         # CSS стили сайта
├── js/             # Скрипты JavaScript
└── index.html      # Главная страница сайта
```

## ⚙️ Как запустить проект локально
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ZakharovVlr/Progect-P1.git
   ```
2. Перейдите в папку с проектом.
3. Откройте файл `index.html` в любом современном браузере (или запустите через плагин Live Server в VS Code).
